## School Days - CS400

### This is a mock HCM that tries to emulate the base functionality of Workday. 
### You will be able to access an online version here without downloading anything here: https://mattchang8.github.io/SchoolDays

### If you want to download this repo and run it, you will need to download the following dependencies:
react js, react-router-dom, react-daterange-picker, react-pro-sidebar, npm
### After setting up the code environment, make sure you are in the correct directory. Type 'npm run dev' into the terminal and run the command. The website should open a browser and run in localHost. You are able to alter the profile information in the 'profiles.json' and 'timesheet.json' files.
